Tutorial de Loops em Programação

Aprenda sobre as estruturas de repetição for, while, e do-while.

Estrutura for

O loop for é usado quando o número de iterações é conhecido. Ele contém três partes: inicialização, condição e incremento.

Exemplo 1: Código


for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}

Exemplo 2: Tabuada


for (let i = 1; i <= 10; i++) {
console.log(${numero} x ${i} = ${numero * i});
}

Exemplo 3: Números Pares


                        let numero = 3;
            
for (let i = 2; i <= 10; i += 2) {
let resultado = numero * i;
console.log(`${numero} x ${i} = ${resultado}`);
}

Exemplo 4: Fatorial


let numero = 5;
let fatorial = 1;
            
for (let i = 1; i <= numero; i++) {
fatorial *= i;
}
            
console.log(`O fatorial de ${numero} é: ${fatorial}`);

Exemplo 5: Contagem Regressiva 


let numeroInicial = 10;

for (let i = numeroInicial; i >= 1; i--) {
console.log(i);
}
console.log('Contagem regressiva concluída!');

Exemplo 6: Multiplicação em Intervalo


let numero = 3;
            
for (let i = 1; i <= 10; i++) {
let resultado = numero * i;
console.log(`${numero} x ${i} = ${resultado}`);
}

Estrutura while

            
O loop while é usado quando o número de iterações depende de uma condição que pode mudar dinamicamente.

            

                
Exemplo 1: Contagem Crescente

                

let i = 1;
            
while (i <= 10) {
console.log(i);
i++;  // Aumenta o valor de "i" em 1 a cada iteração
}




            

                
Exemplo 2: Soma Acumulada

                

let soma = 0;
let i = 1;
                    
while (i <= 5) {
soma += i;  // Soma o valor de i a cada iteração
i++;        // Aumenta o valor de i em 1 a cada iteração
}
                

            

            

                
Exemplo 3:Sequência de Fibonacci 

                

let n1 = 0, n2 = 1, nextTerm;
let count = 1;
let resultado = `${n1} ${n2} `;

while (count <= 8) {
nextTerm = n1 + n2;  // Calcula o próximo número da sequência
resultado += nextTerm + " ";  // Adiciona o próximo número à sequência
n1 = n2;  // Atualiza n1 para o valor de n2
n2 = nextTerm;  // Atualiza n2 para o próximo termo calculado
count++;  // Aumenta o contador
}
                

            

            

                
Exemplo 4: Números Ímpares

                

let max = 20; // Número máximo até onde queremos gerar os ímpares
let i = 1; // O primeiro número ímpar
                    
while (i <= max) {
if (i % 2 !== 0) { // Verifica se o número é ímpar
console.log(i);
}
i++; // Incrementa para o próximo número
}
                

            

            

                
Exemplo 5: Divisores

                

let numero = 36; // Número para o qual queremos encontrar os divisores
let i = 1; // Começamos com 1
                    
while (i <= numero) {
if (numero % i === 0) { // Verifica se 'i' é um divisor de 'numero'
console.log(i); // Exibe o divisor
}
i++; // Incrementa para o próximo número
}

Estrutura do-while

            
O loop do-while executa o bloco pelo menos uma vez antes de verificar a condição.

            

                
Exemplo 1: Soma de Números

                

let soma = 0;
let i = 1;
do {
soma += i;
i++;
} while (i <= 5);
console.log(soma);  // Resultado: 15


            


            

                
Exemplo 2: Soma dos Dígitos de um Número

                

let numero = 1234;
let soma = 0;
do {
soma += numero % 10;
numero = Math.floor(numero / 10);
} while (numero > 0);
console.log(soma);  // Resultado: 10 (1 + 2 + 3 + 4)
                

            


            

                
Exemplo 3: Quadrados dos Números de 1 a 5

                

let i = 1;
do {
console.log(i * i);  // Exibe o quadrado de i
i++;
} while (i <= 5);
                

            


            

                
Exemplo 4: Multiplicação Contínua

                

let multiplicacao = 1;
let i = 1;
do {
multiplicacao *= i;
i++;
} while (i <= 5);
console.log(multiplicacao);  // Resultado: 120 (1 * 2 * 3 * 4 * 5)
                

            


            

                
Exemplo 5: Soma de Números Pares de 1 a 10

                

let somaPares = 0;
let i = 2;
do {
somaPares += i;
i += 2;  // Incrementa 2 para pegar o próximo número par
} while (i <= 10);
console.log(somaPares);  // Resultado: 30 (2 + 4 + 6 + 8 + 10)
